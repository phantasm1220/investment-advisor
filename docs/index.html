🔑 Personal Access Token（PAT）の取得方法

GitHubのPATはこのパネルからワークフローを実行するために必要です。

  1. GitHub → Settings → Tokens (classic) → Generate new token を開く
  2. Noteに名前を入力（例: investment-advisor-panel）
  3. Expirationを選択（90 daysを推奨）
  4. repo と workflow にチェックを入れる
  5. 「Generate token」をクリック → 表示されたトークンをコピーして設定画面に貼り付ける

⚠️ トークンは再表示できません。コピーし忘れた場合は再発行してください。
